The Bad Globals Demo Page

Open up Firebug or Chrome Developer Tools

This won't work in Internet Explorer (but it doesn't need to)

This page contains the following JavaScript:

// some variables
var x = 1; // good
y = 2; // bad (always)
window.z = 3; // bad (sometimes)
window[0] = 4; // bad (sometimes)
window.iWantThisOne = 5; // sometimes bad, but we are ok with this one

BADGLOBALS.exclude("iWantThisOne", "jQuery"); // exclude the globals we don't mind as properties
BADGLOBALS.check(); // do the check


console.log("Report is %o", BADGLOBALS.report());
