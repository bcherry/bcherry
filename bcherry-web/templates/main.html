Welcome to the app-engine-patch sample project

This site is a demonstration of app-engine-patch which is an App Engine port of the Django web framework.

Examples

You can visit Django's admin interface. If needed, you can generate an admin user with the username and password "admin".

Let's have a look at our generic views sample (a simple person list with file uploads).

The CSS styling is done via Blueprint CSS. We use a modified version which is adapted to the app-engine-patch media generator.

This sample project includes a port of django-registration. We maintain a list of other open-source projects to get you started faster with App Engine development.
