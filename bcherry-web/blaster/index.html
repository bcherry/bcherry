Jet Blaster - Mach II

About

This game is a partial port of Jet Blaster Mach II, for the TI-89 graphing calculator, which is the first program of reasonable quality Ben Cherry ever wrote. The game was originally written in C, and compiled using TIGCC. This port is written in pure JavaScript, using the HTML5 <canvas> tag, with no external images or other files. Part of the porting effort involved the creation of a new JavaScript library for simulation of a TI graphing calculator, TI.js (only TI-89 simulation is supported at this time).

Controls

Controls are very simple, and reflect the original game controls mapped to a keyboard.

  • [Up], [Down], [Left], [Right] - maneuver
  • [Space] ([2nd]) - fire cannon
